# System Requirements

RadSystems offers a range of options to choose your development stack. However, it's important to note that certain third-party tools and applications needed to set up your local development environment aren't included in the RadSystems Studio Application Installation Setup. Depending on your chosen development stack, it's recommended to install the following development tools, frameworks, and applications separately. 

## Computer System Requirements

- Supported OS: Windows 7, Windows 8, Windows 10, Windows 11
- Minimum of 4GB RAM (8GB or more recommended)
- Quad-core processor or higher for optimal performance
- High-resolution display (Full HD or higher)
- Active internet connection for package management and updates

**Note:** 
- As of now, RadSystems Studio is compatible exclusively with the Windows Operating System (OS), specifically Windows x86/x64 versions (7, 8, 10, 11). 
- If you're working with a different operating system, the solution involves setting up a Virtual Machine (VM) using Virtualization Tools such as Wine, VirtualBox, VMWare etc and installing the supported Windows OS within that VM. This enables you to successfully install RadSystems Studio within the Virtual Machine environment.

## Development Tools

RadSystems Studio offers a comprehensive platform for building dynamic web, mobile, and desktop applications utilizing database technology. However, having certain tools installed on your development machine can significantly enhance your experience during testing, debugging, and idea generation phases of app development. By leveraging these tools alongside RadSystems Studio, you gain access to advanced capabilities that streamline your workflow and enable more efficient creation of innovative software solutions.:

1. **Visual Studio Code** or any other similar code editor.
2. **Android Studio:** An integrated development environment (IDE) tailored for Android app development. It streamlines the process of creating, testing, and debugging Android applications, allowing you to create engaging mobile experiences.
3. **Database Management Client Software:** Software such as phpMyAdmin or any other Windows desktop application that provides a way to create, manage, and manipulate databases on LocalHost. These tools facilitate efficient database administration and manipulation, which is integral to many application development projects.
