# Main Application Toolbar

![rs-maintoolbar.png](/assets/radsystems-ui/rs-maintoolbar.png)

The main toolbar provides quick access to the most important project-level functions in RadSystems Studio. These controls allow you to configure application settings, manage visual assets, define system behavior, and publish your project — all from a centralized location.

Below is a brief overview of each button. Detailed documentation for each feature is covered in separate sections.

## Save Project

Saves all current changes made to the project configuration, design, and settings. It is recommended to save frequently to prevent loss of work during development.
## App Locale

Opens the localization settings where you can configure language and regional preferences for the application. This is used when building multi-language applications.
## App Icons

Allows you to define the icons used throughout the application. These icons appear in navigation menus, pages, and UI elements to improve usability and visual recognition.
## Project Theme

Opens the theme configuration panel where you can control the visual styling of the application, including colors, layout styles, and UI appearance.
## Project Settings

Provides access to core application configuration such as backend options, system behavior, and general project-level parameters. These settings influence how the application functions at runtime.
## Publish Project (F5)

Builds and deploys the application based on the current project configuration. This process generates the application code and prepares it for testing or deployment to a server environment.
