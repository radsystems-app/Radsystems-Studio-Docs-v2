# Auth Pages

![rs-authentication-authpages.png](/assets/authentication/rs-authentication-authpages.png)

The **Auth Pages** button opens a management interface where you can define the accessibility of individual pages within your application.

- **Excluding Pages**: By default, RadSystems Studio protects all pages with authentication. Within the Auth Pages list, you can select specific pages to "exclude." These excluded pages will be accessible to the public without requiring a login.
    
- **Public Access**: This is ideal for landing pages, "About Us" sections, or public data feeds.
