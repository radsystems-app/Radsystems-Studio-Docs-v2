## App Theme

The **Project Theme** page controls the visual appearance of your generated application. It allows you to define the overall design style, color scheme, and layout presentation based on the UI framework selected when the project was created. Through this interface, you can apply pre-built themes, adjust navigation styling, and preview how the application will look — all without modifying CSS manually. This ensures consistent branding and a professional user interface while keeping the design process simple and framework-aligned.

The options available in the **Project Theme** dialog vary depending on the UI framework selected when the project was created. Each framework has its own styling system, theme structure, and configuration controls, so the settings shown here are tailored specifically to that framework. As a result, the layout and available theme options may differ from one project to another based on the chosen UI technology.
