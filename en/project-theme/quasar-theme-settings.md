# Project Theme — Quasar (Vue)

![rs-projecttheme-quasar.png](/assets/project-theme/rs-projecttheme-quasar.png)


## 01 Layout

**Fixed Header:** Keeps the top header visible while page content scrolls.  
**Fixed Left Drawer:** Keeps the left sidebar permanently visible.  
**Left Drawer Below Header:** Positions the left drawer beneath the header.  
**Left Drawer Below Footer:** Controls whether the left drawer extends below the footer.  
**Fixed Right Drawer:** Keeps the right-side panel fixed.  
**Right Drawer Below Header:** Places the right drawer below the header.  
**Right Drawer Below Footer:** Determines whether the right drawer extends below the footer.  
**Fixed Footer:** Keeps the footer visible at all times.

---

## 02 Theme

**Dark Theme:** Switches the application to dark mode styling.  
**Primary Color:** Main brand color used across the interface.  
**Secondary Color:** Supporting color for UI elements.  
**Accent Color:** Used for emphasis and interactive highlights.  
**Positive Color:** Indicates success states.  
**Negative Color:** Represents errors or critical alerts.  
**Warning Color:** Used for caution messages.  
**Info Color:** Represents informational states.  
**Dark Color:** Used for darker UI areas.  
**Body Color:** Defines the main page background color.

---

## 03 Header

**Include Header:** Shows or hides the header.  
**Header Background Color:** Sets the header background color.  
**Header Background Glossy:** Applies a glossy visual effect.  
**Header Text Color:** Defines header text contrast.  
**Header Elevated:** Adds shadow depth to the header.  
**Header Bordered:** Adds a border around the header.

---

## 04 Footer

**Include Footer:** Shows or hides the footer.  
**Footer Background Color:** Sets footer background styling.  
**Footer Background Glossy:** Applies a glossy effect.  
**Footer Text Color:** Defines footer text contrast.  
**Footer Elevated:** Adds shadow elevation.  
**Footer Bordered:** Applies border styling.

---

## 05 Left Drawer

**Left Drawer:** Enables the primary sidebar.  
**Left Drawer Dense:** Reduces spacing for compact display.  
**Left Drawer Default Display:** Determines whether the drawer is open by default.  
**Left Drawer Mini Mode:** Enables collapsed mini-sidebar behavior.  
**Left Drawer Width:** Sets sidebar width.  
**Left Drawer Min Width:** Defines minimum sidebar width.  
**Left Drawer Bordered:** Adds borders.  
**Left Drawer Elevated:** Adds shadow depth.  
**Left Drawer Overlay:** Enables overlay behavior on smaller screens.  
**Left Drawer Background Color:** Sets sidebar background color.  
**Left Drawer Background Glossy:** Applies gloss effect.  
**Left Drawer Text Color:** Defines sidebar text color.  
**Left Drawer Icon Color:** Controls icon colors.  
**Left Drawer Active Link Background Color:** Highlights the selected menu item.  
**Left Drawer Active Link Text Color:** Defines active item text color.

---

## 06 Right Drawer

**Right Drawer:** Enables a secondary sidebar.  
**Right Drawer Elevated:** Adds shadow depth.  
**Right Drawer Bordered:** Applies borders.  
**Right Drawer Overlay:** Enables overlay mode.  
**Right Drawer Width:** Defines its size.

---

## 07 Input Field

**Input Border Radius:** Adjusts input corner rounding.  
**Input Filled:** Uses filled-style inputs.  
**Input Outlined:** Uses outlined-style inputs.  
**Input Dense:** Makes inputs more compact.  
**Input Rounded:** Applies rounded corners.  
**Input Clearable:** Adds a clear button inside inputs.  
**Input Square:** Removes corner rounding.

---

## 08 Button

**Button Caps:** Forces uppercase button text.  
**Button Glossy:** Applies glossy styling.  
**Button Outline:** Uses outline-style buttons.  
**Button Rounded:** Applies rounded corners.  
**Button Border Radius:** Controls button rounding.  
**Button Push:** Adds push-style depth effect.  
**Button Unelevated:** Removes shadow.  
**Button Flat:** Uses flat-style buttons.

---

## 09 Card

**Card Bordered:** Adds borders to cards.  
**Card Border Radius:** Adjusts card corner rounding.  
**Card Flat:** Removes elevation.  
**Card Shadow Class:** Defines the shadow depth style applied to cards.

---

## Applying the Theme

Click **OK** to apply the selected theme settings to the project. The new visual style will take effect the next time the application is published.
